# Gym Trainer Website
Gym Trainer is a fully responsive fitness and personal trainer website.
Responsive on all devices, built using HTML, CSS, and PHP for basic backend form handling.

➥ Live Demo (http://127.0.0.1:5500/index.html)

🛠️ Technologies Used
HTML – For structuring all pages

CSS – For styling, layout, and responsive design

JavaScript – (Optional interactivity like navigation)

PHP – For handling the contact form submission

🚀 How to Use
Open the index.html file in your preferred browser.

Navigate through pages: About, Courses, Pricing, Gallery, Contact.

To enable the contact form, run the project on a PHP-supported server (like XAMPP).

Customize text, images, and styles by editing .html and .css files.

📌 Pages Included
index.html – Homepage

about.html – About the trainer (Jacson)

courses.html – Training programs: bodybuilding, weight loss, etc.

pricing.html – Membership pricing

gallery.html – Gym and client images

contact.html – Form to connect

contact_process.php – Processes form submission
